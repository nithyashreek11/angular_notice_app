# angular_notice_app
Institute Notice Board — A lightweight Angular standalone-app for posting, searching, filtering and managing institute notices (CRUD + category, importance, validity).

NoticeBoardApp

A lightweight Angular Notice Board application for institutes and academies. The app allows admins to post notices, mark them important, set validity dates, and users to search and filter notices by category. Built with standalone components and Angular Signals for simple state management.
Features

    Add, edit and delete notices (CRUD)
    Category filtering and full-text search
    Mark notices as Important and show important-count statistics
    Validity period (display expired notices differently)
    Responsive UI (desktop and mobile-friendly)
    No backend required — in-memory data service (easy to extend to API)
    Built with Angular standalone components and Signals (Angular 16/17+ compatible)
